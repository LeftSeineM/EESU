# EE Student Union

清华大学电子系学生会活动记录与展示网站。

## 内容定位

- 展示文艺中心、体育中心、联络中心与权益发展相关活动
- 保留原 EELL 联络中心网站入口
- 记录公开资料来源，方便后续补充无线之声公众号文章和活动原图

## 技术栈

- Next.js / React
- TypeScript
- Tailwind CSS
- lucide-react

## 本地运行

```bash
npm install
npm run dev
```

## 静态导出

```bash
npm run build
```

项目已配置 GitHub Pages 工作流，推送到 `main` 后会自动导出并发布 `out/`。
